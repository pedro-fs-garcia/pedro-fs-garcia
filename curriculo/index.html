Principais Projetos

InsightFlow – Plataforma de Análise Preditiva de Comércio

Objetivo: Automatizar a análise e previsão de tendências de comércio exterior a partir de dados públicos para identificar oportunidades de mercado.

  • Ação: Desenvolvi um pipeline de Big Data compreendendo uma API REST (Flask) para ingestão, processamento e consulta de grandes volumes de dados; modelos de séries temporais (StatsModels) para previsão; arquitetura orquestrada com Docker e deploy em ambiente AWS.

Stack: Flask, Pandas, PostgreSQL, Redis, Docker, AWS.

IdScan – Processamento de Documentos com IA Local

Objetivo: Automatizar a extração de dados de RGs via OCR e IA (LLM) rodando localmente para garantir privacidade e otimizar cadastros.

  • AÇÃO: Desenvolvi uma solução em Java que orquestra a extração de texto (Tesseract), a interpretação estruturada com IA (Ollama) e a persistência em banco de dados MySQL.

Stack: Java, Ollama, MySQL, TesseractOCR.

FilmMash

Objetivo: Criar um sistema interativo de classificação de filmes (estilo "FaceMash") com um sistema de ranking em tempo real.

  • Ação: Desenvolvi o backend (Flask) para servir os dados e processar a lógica de pontuação (algoritmo ELO), e construí o aplicativo cliente nativo em Kotlin que consome a API REST.

Stack: Flask, Kotlin, MySQL, SQLite, REST API.

JurisBot – Assistente Jurídico com IA Generativa

Objetivo: Automatizar tarefas de pesquisa, análise e síntese de informações em bases de dados jurídicas usando Agentes Autônomos (LLMs).

  • Ação: Orquestrei um pipeline de agentes de IA (SmolAgents) para executar tarefas complexas de pesquisa e resumo, integrando com modelos de linguagem (Hugging Face) para processamento de texto.

Stack: Python, Hugging Face, Smol-Agents, Gradio.
