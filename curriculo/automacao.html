Principais Projetos

JurisBot – Automação de Pesquisa Jurídica com IA

IdScan – Automação de Extração de Dados de Documentos

Solução que automatiza a extração de dados de RGs via OCR e IA (LLM) rodando localmente, otimizando o processo de cadastro e garantindo privacidade.

  • Desenvolvi a orquestração do fluxo: leitura da imagem (Tesseract), interpretação com IA (Ollama) e estruturação dos dados.
  • Implementei a persistência dos dados extraídos de forma organizada em banco de dados MySQL.

Stack: Java, Ollama, MySQL, TesseractOCR.

InsightFlow – Plataforma de Análise Preditiva de Comércio

Solução de Big Data para analisar e prever tendências de comércio exterior a partir de dados públicos.

  • Desenvolvi a API REST (Flask) para ingestão, processamento e consulta de grandes volumes de dados.
  • Implementei modelos de séries temporais (StatsModels) para gerar previsões de mercado.
  • Orquestrei a arquitetura com Docker e realizei o deploy em ambiente AWS.

Stack: Flask, Pandas, PostgreSQL, Redis, Docker, AWS.

FilmMash

Solução full stack com API REST e cliente Android nativo para classificar filmes de forma interativa usando um sistema de ranking.

  • Implementei o algoritmo de classificação ELO para ranquear os filmes com base nas escolhas dos usuários.

Stack: Flask, Kotlin, MySQL, SQLite, REST API.
